## SSHの設定

Secure SHell (SSH) の設定を行います．

### SSHサーバの起動

```text
$ sudo systemctl start ssh
```

さらに，Raspberry Piが起動した際にSSHサーバが自動的に起動する

```text
$ sudo systemctl enable ssh
```

### マスタ

```text
$ ssh-kegen
```

```text
$ ssh-copy-id 163.221.190.225
$ scp ~/.ssh/id_rsa 163.221.190.225:~/.ssh/id_rsa
```
